Kreditkarten Validierung in Wordpress

== Description ==

Mit dem Plugin Kreditkarten Validierung ist es m&ouml;glich Kreditkartennummern auf ihre G&uuml;ltigkeit zu &uuml;berpr&uuml;fen. Einfach Integration in einen bestehenden Blog m&ouml;glich. Support: [http://www.kreditkarte.org/](http://www.kreditkarte.org/)

== Installation ==

1. Extract `kreditkarten-validierung-1.0.zip`.
2. Upload the `kreditkarten-nachrichten`-folder to your `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Add the shortcode <!-- kreditkartenvalidierung --> in your post
